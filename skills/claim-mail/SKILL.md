## Background

你正在原神游戏主界面中操作。游戏已启动并加载完毕，角色站在游戏世界中。

### 邮件系统
邮件入口在**派蒙菜单**内，不在主界面 HUD 上。必须先**按 Esc 键**呼出派蒙菜单才能访问。

#### 派蒙菜单布局
在游戏主界面空闲状态下**按 Esc 键**打开派蒙菜单，鼠标光标出现，可以正常点击：
- **左侧纵列小图标**：从上到下排列的快捷图标，**第 3 个是邮件图标**（信封形状）。有新邮件时会显示红点。

#### 邮箱界面布局
点击邮件图标后进入：
- **顶部**：左侧"邮箱"标题，右侧显示"邮件数 X/1000"和关闭按钮（✕）。
- **左侧面板**：邮件列表，每封邮件显示图标、标题、发件人、过期时间。
- **右侧面板**：选中邮件的详情（发件人、时间、正文、附件物品图标）。
- **底部左侧**：**「全部领取」按钮**（绿色图标）——一键领取所有邮件附件。
- **底部中间**：「删除已读」按钮（红色图标）。
- **底部右侧**：⭐收藏按钮 + 「领取」按钮（单封领取）。

## Goal

打开邮箱，点击「全部领取」一键领取所有邮件附件。

操作步骤：
1. 在游戏主界面空闲状态下，**按 Esc 键**呼出派蒙菜单（主界面鼠标锁定，只能用键盘操作）。
2. 派蒙菜单打开后鼠标解锁可用，在**左侧纵列小图标**中，找到**从上往下第 3 个图标**（信封形状的邮件图标），点击打开邮箱。
3. 进入邮箱后，查看底部按钮区域。
4. 点击**底部左侧**的**「全部领取」按钮**（绿色图标），一键领取所有附件。
5. 如果弹出物品展示窗口，点击空白处关闭。
6. 关闭邮箱界面（点击右上角 ✕），确认回到游戏主界面，任务完成。

如果邮箱内没有可领取的邮件（「全部领取」按钮不可点击或邮件列表为空），直接关闭并报告 done success。

## Known Issues

- 派蒙菜单打不开——可能当前在对话、剧情或其他界面中，先按 Escape 退出当前界面，回到主界面空闲状态后再按 Esc 呼出菜单。
- 按 Esc 没反应——游戏画布可能未获得焦点，先点击游戏画面中央激活画布，再按 Esc。
- 邮件图标没有红点——说明没有新邮件，但仍应打开邮箱确认，因为可能有未领取附件的已读邮件。
- 「全部领取」按钮灰色不可点击——所有邮件附件已领取，关闭并报告 done success。
- 领取后弹出物品展示弹窗——点击空白处或确认按钮关闭。
- 邮件列表需要滚动——「全部领取」会领取所有邮件，无需逐封滚动。
- 误点云游戏侧边栏弹出浮层遮挡邮箱——点击浮层外的游戏区域或按 Escape 关闭浮层，再继续操作。注意邮件图标在派蒙菜单内，不在屏幕最左侧的云游戏侧边栏上。
- 打开邮箱后界面加载缓慢——选择 wait 等待加载完成。
